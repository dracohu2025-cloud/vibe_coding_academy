# Google Antigravity: 代码摆脱引力

> "IDE 已经死了。平台万岁。"

**[[[[Antigravity]]]]** (反重力) 是 Google 推出的下一代开发平台。它不是一个编辑器，它是一个**云端 Agent 编排系统**。

## 核心概念

### 1. 云端原生

你的代码、你的环境、你的 Agent，全都在云端。
你在本地只需要一个轻量级的客户端（甚至只是一个浏览器 Tab）。

### 2. Agent 战队 (Swarm)

在 [[[[Antigravity]]]] 里，你不是一个人在战斗。
你可以召唤一个 **Agent 战队**：

* **Architect Agent**: 负责看需求，拆解任务。
* **Coder Agent**: 负责写代码。
* **Tester Agent**: 负责写测试用例并运行。
* **Reviewer Agent**: 负责审查代码。

作为人类，你变成了 **"任务指挥官" (Mission Commander)**。你只负责下达战略目标，剩下的通过 Agent 之间的协作自动完成。

### 3. Artifacts (产物)

Agent 之间的交流不再是文字，而是 **Artifacts**（产物）。
架构师 Agent 会产出一个 "设计文档 Artifact"，直接传给 程序员 Agent；程序员 Agent 产出 "代码 Artifact"。
所有的过程都是透明的、可追溯的。
