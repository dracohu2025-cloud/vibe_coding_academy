# [[Cursor]]: AI 代码编辑器的现任霸主

> "它不仅是一个编辑器，它是你的分身。"

**[[Cursor]]** 是目前体验最成熟、生态最完善的 AI Native 编辑器。基于 VS Code 二次开发，让你能够无缝迁移，但又拥有完全不同的 AI 体验。

## 核心特性

### 1. Tab (Copilot++)

[[Cursor]] 的补全（Tab）不仅仅是补全一行代码。它能预测你的光标移动，预测你下一块要改哪里。
当你写了一个变量名，它可能会直接帮你把后面几行的逻辑全写完。这种 "Flow" (心流) 体验是它最大的护城河。

### 2. Composer (作曲家)

这是一个多文件编辑模式 (Cmd+I)。
你可以告诉它："帮我把这个 API 的返回字段改了，然后更新所有的前端组件类型定义。"
它会同时打开后端文件、前端文件、Type 文件，一起修改。这在以前是不可想象的。

### 3. codebase Indexing

它会对你的本地代码库进行深度索引。当你问 "auth 逻辑在哪里？" 时，它给出的答案是基于你真实代码的，而不是瞎猜。
