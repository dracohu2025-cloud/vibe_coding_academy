# 全能驾驶舱：Tools of Trade (2025版)

> "工欲善其事，必先利其器。在 2025 年，你的'器'不仅是工具，而是你的'僚机'团队。"

欢迎来到 **Vibe Coding 武器库 2.0**。
世界变化太快。几个月前的"神器"现在可能已经成了"古董"。我们不怀旧，我们要最强的。

---

## 1. 核心指挥所：AI IDEs

VS Code 依然是地基，但我们需要在上面盖摩天大楼。

### **Cursor** (现任霸主)

依然是目前体验最丝滑的 AI 编辑器。

- **Composer**: 它的多文件编辑能力依然是标杆。
- **Cmd+K**: 肌肉记忆。
- **优势**: 极其成熟，生态完善。

### **Trae** (强力挑战者)

来自字节跳动的黑马，完全免费且极其强大。

- **Solo Mode**: 这是一个"全自动"模式。你给它一个需求，它不仅写代码，还负责规划、测试、运行。它就像一个不需要睡觉的高级工程师。
- **原生中文友好**: 对中文语境的理解甚至优于 Cursor。

---

## 2. 重火力支援：Lethal Models

模型就是你的弹药。2025年的弹药库里只有两个名字值得你记住。

### **Gemini 3.0 Pro** (Vibe Coding 之神)

Google 的最新力作。

- **特长**: **Vibe Coding** 的定义者。它不仅懂代码，更懂"感觉"。它的多模态能力（看图、看视频）无敌。
- **用途**: 创意落地、前端设计、从草图生成 App。

### **Claude 4.5 Sonnet** (逻辑怪兽)

Anthropic 的当家花旦。

- **特长**: 逻辑缜密，不仅写代码，还能思考架构。它的 "Thinking" 模式能进行深度推理。
- **用途**: 复杂后端逻辑、重构、系统架构设计。

---

## 3. 特种部队：Agentic Swarm

当 IDE 里的 Chat 窗口已经满足不了你时，你需要呼叫**特种部队**。这些不是简单的聊天机器人，它们是能够**自主执行任务**的智能体。

### **Google Antigravity** (旗舰级)

这不是一个工具，这是一个**平台**。

- **概念**: "让代码摆脱地心引力"。所有的开发环境都在云端。
- **能力**: 你可以指挥整整一个**Agent 团队**。一个负责写测试，一个负责写实现，一个负责审查。这是"一人公司"的终极形态。

### **Claude Code** (潜行者)

运行在终端里的幽灵 (CLI Tool)。

- **操作**: 没有 UI，只有命令行。你给它权限，它就在你的文件系统里穿梭。
- **场景**: "帮我把整个项目的 Log 库替换成 Pino，并修复所有报错。" —— 它会默默在后台干完这一切，然后提交 PR。

### **Roo Code** (原 Root / 游侠)

VS Code 里的开源英雄。

- **特点**: **Bring Your Own Key**。你直接用 OpenRouter 的 Key，便宜又实惠。
- **模式**: 它有 "Architect" (架构师) 和 "Code" (工程师) 等不同模式，分工明确。

### **Augment Code** (记忆大师)

- **特点**: 它的上下文记忆力是惊人的。当你有一个巨大的旧项目（百万行代码），Augment 是唯一能瞬间理解 "这行代码两年前为什么要这么写" 的 AI。

### **Kiro / Kilo** (严谨派)

- **特点**: **Spec-Driven** (文档驱动)。它们不瞎写，它们先写文档（Spec），再写代码。适合对质量要求极高的工程。
