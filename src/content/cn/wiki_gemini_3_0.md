# Gemini 3.0 Pro: Vibe Coding 之神

> "它不只是理解代码，它理解你的想法。"

**[[Gemini 3.0]] Pro** 是 Google DeepMind 在 2025 年发布的旗舰模型，它彻底改变了我们写代码的方式，并定义了 **"Vibe Coding"** 这一概念。

## 核心特性

### 1. Vibe Coding 定义者

以前的模型需要你写详细的 Spec（规格说明书）。
[[Gemini 3.0]] 不需要。你只需要告诉它："给我做一个像 Spotify 那样的播放器，但是要赛博朋克风格的。"
它会立刻理解你的 "Assert" (审美) 和 "Vibe" (感觉)，直接生成带有完美 CSS 动效的完整应用。

### 2. 原生多模态 (Native Multimodal)

它不仅读文字，还能读像素。

* **截图转代码**：画一个草图，或者截一张别的 App 的图，它能像素级复刻 UI。
* **视频理解**：录一段屏幕操作视频，告诉它 "把这个 Bug 修了"，它能看懂你在哪里点了什么，发生了什么错误。

### 3. 超长上下文 (Infinite Context)

它的上下文窗口极大。你可以把整个 [[React]] 源码库扔给它，问它某个底层 Hook 是怎么实现的。它不会幻觉，它会像一个在这行干了 10 年的老工程师一样给你解释。

## 最佳食用方法

* **IDE**: 配合 **Google [[Antigravity]]** 或 **[[Cursor]]** 使用。
* **Prompt 技巧**: 它是 "直觉型" 选手。不要给它太死板的指令，多用形容词描述你想要的感觉。
