# [[Kilo]] Code: The Visual Builder

> "What You See Is What You Get."

**[[Kilo]] Code** attempts to bridge "Code" and "Vision".

## Core Features

### 1. App Builder

It has a built-in Visual App Builder. Drag components, and AI generates high-quality code in the background.
Conversely, edit code, and the UI updates. Real two-way binding.

### 2. Cloud Environment

It runs in cloud containers. Environment (Node.js, Python, Docker) is pre-configured. Open browser, start coding.

## Verdict

**"For indie hackers wanting quick prototypes, [[Kilo]] is the fastest starting line."**
